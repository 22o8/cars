# AutoDealer Pro Web PWA V7

هذه النسخة مبنية على آخر ملف مرفوع، وتم فيها:

- دمج صفحات السيارات الزائدة داخل صفحة السيارات الرئيسية لتقليل التعقيد وتسريع النظام.
- إضافة رفع أكثر من صورة لكل سيارة.
- إضافة تعديل وحذف السيارات مع حماية حذف السيارة المرتبطة ببيع.
- توضيح كل حقول السيارات بتسميات وتعليمات فوق كل خانة.
- تبسيط إدارة العملاء وإبقاء صفحة العملاء الأساسية مع مستمسكات العميل فقط.
- إضافة نسخة احتياطية، استعادة نسخة، وإعادة ضبط المصنع من صفحة واحدة.
- إبقاء نظام الصلاحيات للموظفين حسب الأقسام.
- تحسين القائمة الجانبية وحذف الروابط غير الضرورية.

بعد استبدال النسخة:

```powershell
npm install
npx prisma db push
npm run dev
```

ملاحظة: إذا كانت قاعدة البيانات قديمة، نفذ `npx prisma db push` حتى يضاف حقل صور السيارات.
