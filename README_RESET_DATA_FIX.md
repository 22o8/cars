# تعديل مسح البيانات

هذه النسخة تصلح زر مسح البيانات حتى يحذف كل البيانات التشغيلية بالكامل، بما فيها:

- سجلات الشراء
- سجلات البيع
- الأقساط والدفعات
- الفواتير
- الخزنة والمصاريف
- العملاء ومستمسكاتهم
- السيارات والمخزن
- العروض والتصنيفات
- سجلات النسخ الاحتياطي وسجل العمليات
- حسابات الموظفين غير المدير

ويتم الإبقاء على حسابات المدير فقط وإعدادات النظام الأساسية.

ملاحظة: لا يتم حذف ملفات المشروع ولا يتم عمل Reset لقاعدة البيانات من خارج النظام. المسح يتم من داخل صفحة النسخ الاحتياطي والضبط فقط بعد كتابة رمز التأكيد:

RESET-AUTODEALER
