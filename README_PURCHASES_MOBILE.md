# إضافات نظام الشراء ولوحة الهاتف

تمت إضافة قسم جديد باسم **شراء السيارات** بجانب نظام البيع.

## ما تمت إضافته

- صفحة `/purchases` لتسجيل شراء سيارة.
- حقول الشراء: اسم البائع، رقم الهاتف، اسم السيارة، الشركة، النوع، الموديل، مبلغ الشراء، الواصل، الباقي، من تاريخ، المدة، تاريخ الاستحقاق، صور السيارة/العقد.
- حساب الباقي تلقائياً.
- إضافة السيارة إلى المخزن تلقائياً عند تسجيل الشراء إذا تم اختيار ذلك.
- تسجيل الواصل كعملية صرف من الخزنة.
- ربط الشراء بالتقارير والنسخ الاحتياطي ولوحة التحكم.
- تحسين الصفحة الرئيسية للهاتف لتعرض:
  - سجل السيارات المباعة والباقي من أقساطها.
  - سجل السيارات المشتراة والواصل والباقي من دينها.

## مهم بعد الرفع

لأن قاعدة البيانات صار بها جدول جديد `Purchase`، نفذ بعد الرفع أو محلياً:

```bash
npx prisma db push
```

ثم أعد نشر المشروع على Vercel.
