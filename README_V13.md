# AutoDealer Pro V13

تم تحسين النسخة للاستخدام على الهاتف والحاسوب:

- تثبيت الجلسة بعد Refresh وعدم الرجوع إلى صفحة تسجيل الدخول عند وجود Cookie صحيح.
- تحسين تمرير الصفحات والجداول على الهاتف.
- إضافة قائمة جانبية للهاتف مع شريط سفلي سريع.
- تحسين حجم الأزرار والحقول حتى تكون مناسبة للمس.
- تقليل مشاكل Hydration بين SSR والعميل.

أوامر التشغيل:

```powershell
npm install
npx prisma db push
npm run dev
```
