# التنفيذ السريع للبيع والشراء

تم تبسيط الصفحة الرئيسية حتى تكون مناسبة للهاتف والاستخدام اليومي في المعرض.

## الموجود الآن

- أول ما يدخل المستخدم تظهر أمامه بطاقة التنفيذ السريع.
- يستطيع اختيار:
  - بيع سيارة
  - شراء سيارة
- الحقول المختصرة:
  - اسم صاحب العملية
  - اسم السيارة
  - الواصل
  - الباقي
  - المدة بالأيام أو الأشهر
  - من تاريخ
  - رقم الهاتف اختياري
  - المستمسكات/الصور اختيارية
- أسفل الصفحة تظهر قائمتان:
  - السيارات التي تم بيعها مع الواصل والباقي
  - السيارات التي تم شراؤها مع الواصل والباقي

## قاعدة البيانات

تمت إضافة API جديد فقط، بدون كسر الجداول الحالية:

- `POST /api/quick/sale`
- `POST /api/quick/purchase`

نظام الشراء يستخدم جدول `Purchase` الموجود.
نظام البيع ينشئ العميل والسيارة والبيع والقسط والفاتورة تلقائياً.
