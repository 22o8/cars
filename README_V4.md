# AutoDealer Pro Web PWA V4

نسخة ويب محسنة من نظام AutoDealer Pro.

## التحديثات

- تصميم داكن فاخر قريب من الواجهة المطلوبة.
- ثيمان: داكن وفاتح.
- إزالة الإيموجيات من الواجهة واستبدالها بأيقونات خطية داخلية.
- توسيع القائمة الجانبية بنفس أقسام النسخة القديمة:
  - إدارة السيارات: جميع السيارات، إضافة سيارة، الفئات والماركات، حالات السيارات، المواصفات، الصور.
  - إدارة المبيعات: المبيعات، الأقساط والدفعات، الفواتير، المدفوعات، العروض والخصومات، المصاريف.
  - إدارة العملاء: جميع العملاء، إضافة عميل، فئات العملاء، سجل التواصل.
  - التقارير والإحصائيات.
  - إعدادات النظام والحساب والنسخ الاحتياطي.
- إضافة جداول Prisma جديدة للفئات والماركات وفئات العملاء وسجل التواصل والعروض والنسخ الاحتياطي.

## التشغيل

```powershell
npm install
npx prisma db push
npm run db:seed
npm run dev
```

مهم: لا ترفع ملف `.env` إلى GitHub.
