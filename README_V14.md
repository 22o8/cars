# AutoDealer Pro Web PWA V14

التحديثات:
- تحسين واجهة الهاتف بالكامل.
- نقل الصفحات المهمة إلى شريط قوائم علوي للهاتف مع زر المزيد.
- إبقاء شريط تنقل سفلي مختصر للهاتف.
- إضافة زر تثبيت التطبيق حتى يعمل كـ Add to Home Screen عند دعم المتصفح.
- إضافة طلب صلاحية الإشعارات من الواجهة.
- تنبيه تلقائي للأقساط المستحقة أو المتأخرة أثناء فتح التطبيق أو تشغيله كـ PWA.
- تحسين صفحة الإشعارات وشرح حالة صلاحية الجهاز.

ملاحظة مهمة: إشعارات الويب تعتمد على سماح المتصفح. على iPhone يجب إضافة الموقع إلى الشاشة الرئيسية حتى تكون تجربة PWA أفضل.
