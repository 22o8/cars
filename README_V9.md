# AutoDealer Pro Web PWA V9

إصلاحات هذه النسخة:
- إصلاح ربط الأقساط والدفعات مع المبيعات والخزنة والفواتير.
- إنشاء قسط تلقائي لأي عقد فيه مبلغ متبقي ولم تُنشأ له أقساط سابقاً.
- زر إصلاح ربط الأقساط داخل صفحة الأقساط والدفعات.
- تحديث حالة الأقساط المتأخرة تلقائياً.
- تحسين صفحة إعدادات النظام والحساب والنسخ الاحتياطي.
- إبقاء الصفحات الأساسية فقط في القائمة.

أوامر التشغيل:

```powershell
npm install
npx prisma db push
npm run db:seed
npm run dev
```

ملاحظة: يجب وضع DATABASE_URL و JWT_SECRET داخل ملف .env.
