# AutoDealer Pro Web PWA V8

هذه النسخة مكملة على آخر ملف، وتم التركيز على:

- تنظيف الإعدادات وحذف/دمج الصفحات الزائدة.
- إكمال إدارة السيارات داخل صفحة واحدة مع رفع الصور والتعديل والحذف.
- إكمال إدارة العملاء مع رفع أو التقاط صور المستمسكات.
- ربط الأقساط والدفعات فعلياً بالمبيعات والخزنة والفواتير.
- تسجيل دفعة كاملة أو جزئية لكل قسط.
- إنشاء سند قبض قسط تلقائياً عند الدفع.
- جعل سند القبض يزيد الخزنة وسند الصرف ينقصها.
- تحسين التقارير ومنع تداخل الأرقام الكبيرة.
- فواتير احترافية يمكن عرضها وطباعتها وحفظها PDF من نافذة الطباعة أو Word.

## التشغيل

```powershell
npm install
npx prisma db push
npm run db:seed
npm run dev
```

## ملاحظة مهمة
لا ترفع ملف `.env` إلى GitHub لأنه يحتوي رابط قاعدة البيانات.
