# AutoDealer Pro V18

تم تعديل آلية سعر الصرف لتعمل بالطريقة المطلوبة:

- في إعدادات النظام يوجد الآن قسم: سعر صرف الدولار اليوم.
- تكتب مثلاً: 100 دولار = 150000 دينار عراقي.
- النظام يحسب تلقائياً سعر الدولار الواحد = 1500 دينار.
- كل الإحصائيات والتقارير والخزنة والأرباح والديون التي تحتاج تحويل الدولار إلى الدينار تستخدم السعر الجديد مباشرة.
- العقود والفواتير تبقى محفوظة بعملتها الأصلية حتى تبقى البيانات دقيقة محاسبياً.

بعد الرفع على Vercel تأكد من وجود DATABASE_URL و JWT_SECRET في Environment Variables.
