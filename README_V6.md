# AutoDealer Pro Web PWA V6

نسخة محسنة جاهزة للتجربة تشمل:
- تصميم داكن وفاتح.
- تحسين الربط بين الصفحات والبيانات.
- فواتير احترافية قابلة للطباعة وحفظ PDF من نافذة الطباعة.
- حفظ Word للفواتير.
- طباعة كل الفواتير دفعة واحدة.
- رفع/التقاط صور مستمسكات العملاء.
- تعديل بروفايل المستخدم وصورة البروفايل.
- إدارة موظفين بصلاحيات فعلية على الصفحات.
- نسخ احتياطي JSON.
- إعادة ضبط مصنع برمز تأكيد.

بعد فك الضغط:
```powershell
npm install
npx prisma db push
npm run db:seed
npm run dev
```

مهم: لا ترفع ملف `.env` إلى GitHub.
