# AutoDealer Pro Web PWA V5

تحديث V5 يركز على:

- تحسين سرعة التنقل بين الصفحات باستخدام prefetch و useLazyFetch للصفحات الأساسية.
- تعطيل DevTools افتراضياً وتقليل الانتقالات غير الضرورية.
- إضافة توضيحات فوق حقول الإدخال حتى يكون واضحاً ماذا تكتب في كل خانة.
- تطوير صفحة الموظفين: إضافة موظف، حذف موظف، تفعيل/تعطيل الحساب، وتعديل الصلاحية.
- حماية عمليات الموظفين من جهة السيرفر بحيث تكون للمدير فقط.
- تحسين تجربة التسجيل والإضافة والبيع والفواتير برسائل واضحة وزر تحميل أثناء التنفيذ.

## التشغيل

```powershell
npm install
npx prisma db push
npm run db:seed
npm run dev
```

## ملاحظة مهمة
لا ترفع ملف `.env` إلى GitHub.
