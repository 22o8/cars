# AutoDealer Pro V17

هذه النسخة تركّز على ربط النظام بالكامل بدون زر إصلاح يدوي:

- الأقساط تتزامن تلقائياً عند فتح لوحة التحكم، المبيعات، الأقساط، التقارير، الحسابات، الفواتير، والإشعارات.
- حذف زر إصلاح ربط الأقساط من صفحة الأقساط.
- حساب المدفوع والمتبقي من جميع الدفعات المرتبطة بالعقد.
- عند إنشاء بيع أقساط يتم توزيع المتبقي بدقة، والقسط الأخير يأخذ فرق التقريب.
- عند تسديد قسط يتم تحديث: القسط، العقد، المدفوع، المتبقي، الخزنة، وسند القبض.
- التقارير والحسابات تحسب مبالغ الدولار بالدينار حسب سعر الصرف في الإعدادات.
- منع حذف العميل أو السيارة إذا كانت مرتبطة بعقد بيع.

قبل الرفع:

```powershell
npm install
npx prisma db push
npm run build
```

إعدادات Vercel:

- Build Command: `npx prisma generate && npm run build`
- Output Directory: فارغ
- Install Command: `npm install`
