# AutoDealer Pro Web PWA V10

تم إصلاح صفحات الإعدادات الثلاث:
- إعدادات النظام
- إعدادات الحساب
- النسخ الاحتياطي والضبط

وتم تقوية ربط الأقساط والدفعات:
- فحص كل المبيعات التي تحتوي مبلغاً متبقياً.
- إنشاء قسط مفقود تلقائياً إذا كان البيع مرتبطاً بمبلغ متبقٍ ولا توجد له أقساط.
- تحديث المبلغ المدفوع والمتبقي داخل عقد البيع اعتماداً على الدفعات المسجلة.
- زر إصلاح ربط الأقساط داخل صفحة الأقساط.
- كل دفعة قسط تنشئ حركة خزنة وسند قبض.

بعد فك الضغط:
```powershell
npm install
npx prisma db push
npm run dev
```

إذا لديك بيانات قديمة وفيها مبيعات أقساط لم تظهر:
افتح صفحة الأقساط والدفعات واضغط زر: إصلاح ربط الأقساط.
